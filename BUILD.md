# Инструкции по сборке VPN Server Manager

Этот документ содержит подробные инструкции по созданию инсталляторов и портативных версий VPN Server Manager для Windows и macOS.

## Требования

### Общие требования
- Python 3.7 или выше
- Установленный pip
- PyInstaller (`pip install pyinstaller`)

### Для Windows
- Windows 10/11
- [Inno Setup 6](https://jrsoftware.org/isdl.php) для создания инсталлятора
- Git Bash или PowerShell для запуска скриптов

### Для macOS
- macOS 10.14 или выше
- Xcode Command Line Tools (запустите `xcode-select --install`)
- Утилита `iconutil` (встроена в macOS)
- Утилита `sips` (встроена в macOS)

## Подготовка к сборке

1. Клонируйте или загрузите репозиторий проекта
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   # На Windows:
   venv\Scripts\activate
   # На macOS/Linux:
   source venv/bin/activate
   ```
3. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   pip install pyinstaller
   ```

## Сборка для Windows

### Подготовка иконки
1. Создайте иконку для приложения в формате .ico
2. Поместите файл `favicon.ico` в папку `static/`

### Создание полного инсталлятора
1. Перейдите в корневую директорию проекта
2. Запустите скрипт сборки:
   ```bash
   python build_windows.py
   ```
3. После успешной сборки в папке `dist/` будет создан файл `VPNServerManager_Setup_1.0.0.exe`

### Портативная версия
Скрипт `build_windows.py` автоматически создаст портативную версию приложения. После выполнения скрипта в папке `dist/` вы найдете ZIP-архив `VPNServerManager-Portable.zip`.

## Сборка для macOS

### Подготовка иконки
1. Создайте иконку для приложения в формате .png
2. Поместите файл `icon.png` в папку `static/`
3. Скрипт автоматически создаст `.icns` файл из PNG

### Создание .app и .dmg
1. Перейдите в корневую директорию проекта
2. Запустите скрипт сборки:
   ```bash
   python3 build_macos.py
   ```
3. После успешной сборки в папке `dist/` будет создан `.app` бандл и `.dmg` установочный образ

## Структура портативной версии

Портативная версия содержит следующие элементы:
- Исполняемый файл приложения
- Папка `data/` для хранения данных
- Папка `uploads/` для хранения файлов
- Файл конфигурации `portable_config.ini`
- Автоматически сгенерированный ключ шифрования в `.env`

## Дополнительные замечания

- При распространении полного инсталлятора Windows обратите внимание на возможные предупреждения SmartScreen
- Для macOS может потребоваться нотаризация приложения через Apple Developer Program для распространения
- Портативная версия сохраняет все данные в своей директории, что удобно для переноса между компьютерами

## Решение проблем

### Windows
- Если Inno Setup не найден, вы можете скомпилировать `.iss` файл вручную
- Убедитесь, что у вас есть права на запись в папку `dist/`

### macOS
- Если возникают проблемы с созданием иконки, проверьте наличие Xcode CLI Tools
- Для проблем с DMG используйте альтернативные методы создания образов
