# Changelog v3.3.3 - Улучшенная информация об экспорте

## 📅 Дата выпуска: 17.07.2025

## ✨ Новые возможности

### 💡 Информативное модальное окно для экспорта
- **Интерактивное всплывающее окно с подробной информацией**
  - 📋 Показывает тип экспортируемого файла
  - 📁 Указывает точное место сохранения (`~/Downloads/`)
  - ⚠️ Содержит важные рекомендации для пользователя
  - ✅ Кнопка подтверждения для продолжения экспорта

### 🎯 Улучшения пользовательского опыта
- **Прозрачность процесса**: Пользователь заранее знает, что и куда будет сохранено
- **Предотвращение путаницы**: Четкие инструкции о расположении файлов
- **Современный интерфейс**: Красивое темное модальное окно в стиле приложения

## 🔧 Технические улучшения

### JavaScript функции
- `showExportInfo(exportType)` - показ информации о типе экспорта
- `confirmExport(exportType)` - подтверждение и выполнение экспорта
- `closeExportModal()` - закрытие модального окна
- Автоматическое закрытие при клике вне модального окна

### Кнопки экспорта
- Преобразованы из ссылок `<a>` в кнопки `<button>`
- Интегрированы с системой модальных окон
- Улучшенная визуальная обратная связь

## 📱 Примеры информации в модальном окне

**Экспорт данных серверов:**
```
Экспортируется: Файл данных серверов (.enc)
Место сохранения: ~/Downloads/servers_export_[дата_время].enc
```

**Экспорт ключа шифрования:**
```
Экспортируется: Ключ шифрования (.env)
Место сохранения: ~/Downloads/SECRET_KEY_[дата_время].env
```

**Полный экспорт:**
```
Экспортируется: Полный архив (.zip)
Место сохранения: ~/Downloads/vpn_servers_backup_[дата_время].zip
```
