### Обзор: локализация (i18n) в VPN Server Manager

В этом разделе собраны практические инструкции по добавлению и поддержке переводов в проекте. Вы можете выбрать любой путь:

- Метод 1: Flask-Babel (ручной, классический, продакшн-надёжный)
- Метод 2: Babel CLI (экстракция, обновление, компиляция .po/.mo)
- Метод 3: Автоперевод .po (ускорение первичного наполнения переводов)
- Дополнительно: добавление нового языка, устранение проблем, упаковка в .app

С чего начать:
- Если вы впервые настраиваете переводы: прочитайте «Flask-Babel» и «Babel CLI: рабочий цикл»
- Если тексты уже помечены `_(...)`, а нужно быстро заполнить пропуски: смотрите «Автоперевод .po»
- Если необходимо добавить язык: «Добавить новый язык»
- Если сборка .app не подтягивает переводы: «Упаковка (PyInstaller)» и «Траблшутинг»
