# Управление версиями

Актуальное руководство по версиям перенесено в корневой файл:
- [`../VERSION_MANAGEMENT.md`](../VERSION_MANAGEMENT.md)

Коротко:
- источник правды для версии релиза — `config.json`
- синхронизация выполняется через `tools/update_version.py`
- `config/config.json.template`, `README.md`, `vpn-manager-installer.iss` и fallback-версии в коде обновляются автоматически
