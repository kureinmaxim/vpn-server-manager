# Changelog v3.3.1 - Исправление отображения пути к файлу данных

## 📅 Дата выпуска: 17.07.2025

## ✨ Исправления

### 🔧 Улучшение пользовательского интерфейса
- **Исправлено отображение пути к активному файлу данных в настройках**
  - Теперь показывается полный абсолютный путь вместо относительного
  - При запуске из проекта: `/путь/к/проекту/data/servers.json.enc`
  - При запуске из установленной версии: `~/Library/Application Support/VPNServerManager/data/servers.json.enc`
  - Это помогает пользователю точно знать расположение своих данных

## 🔨 Технические изменения
- Изменен контекстный процессор Flask `inject_service_urls()` 
- Теперь используется функция `get_active_data_path()` для получения полного пути
