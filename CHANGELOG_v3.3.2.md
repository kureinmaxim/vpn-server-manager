# Changelog v3.3.2 - Исправление экспорта файлов

## 📅 Дата выпуска: 17.07.2025

## ✨ Основные исправления

### 🔧 Исправлена проблема с экспортом файлов
- **Решена ошибка "Read-only file system" при создании архивов**
- **Все экспортируемые файлы теперь сохраняются в папку Downloads пользователя**
  - 📁 Экспорт данных (.enc файлы)
  - 🔐 Экспорт ключей (.env файлы) 
  - 📦 Полный экспорт (ZIP архивы)

### 🚀 Улучшения пользовательского опыта
- **Безопасное место сохранения**: Все файлы экспортируются в `~/Downloads/`
- **Понятные уведомления**: Сообщения указывают точное место сохранения
- **Автоматический fallback**: Если папка Downloads недоступна, используется директория приложения

## 🔨 Технические изменения
- Добавлена функция `get_export_dir()` для определения безопасной директории экспорта
- Заменены все вызовы `os.getcwd()` на `get_export_dir()` в функциях экспорта:
  - `export_data()` - экспорт файлов данных
  - `export_key()` - экспорт ключей шифрования
  - `export_package()` - создание полного архива

## 📱 Примеры экспортируемых файлов
```
~/Downloads/servers_export_20250717_062345.enc
~/Downloads/SECRET_KEY_20250717_062345.env
~/Downloads/vpn_servers_backup_20250717_062345.zip
```
